# Tech Blog Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build and deploy a minimal Astro tech blog to GitHub Pages with native SVG/HTML rendering and a post-type filter.

**Architecture:** Astro static site scaffolded from the official blog starter, extended with MDX support and a vanilla-JS type filter on the index page. GitHub Actions builds and deploys on every push to `main`.

**Tech Stack:** Astro 5, @astrojs/mdx, Zod (content schema), GitHub Actions, GitHub Pages

import BaseHead from '../components/BaseHead.astro';

const posts = (await getCollection('blog')).sort(
  (a, b) => b.data.pubDate.valueOf() - a.data.pubDate.valueOf()

---

<!doctype html>
<html lang="en">
  <head>
    <BaseHead title={SITE_TITLE} description={SITE_DESCRIPTION} />
    <style>
      .filter-tabs {
        display: flex;
        gap: 0.5rem;
        margin-bottom: 2rem;
        flex-wrap: wrap;
      }
      .filter-tabs button {
        padding: 0.3rem 0.9rem;

        font-size: 0.875rem;

        color: #fff;
        border-color: var(--accent, #4f46e5);
      }
      .post-item[hidden] { display: none; }
    </style>
  </head>
  <body>
    <Header />
    <main>
      <section>
        <div class="filter-tabs">
          <button class="active" data-filter="all">All</button>
          <button data-filter="article">Article</button>
          <button data-filter="showcase">Showcase</button>
          <button data-filter="experiment">Experiment</button>
        </div>
        <ul>
          {posts.map((post) => (
            <li class="post-item" data-type={post.data.type}>
              <a href={`/blog/${post.slug}/`}>
                <img width={720} height={360} src={post.data.heroImage} alt="" />
                <h4 class="title">{post.data.title}</h4>

              </a>
            </li>

      const buttons = document.querySelectorAll('.filter-tabs button');
      const posts = document.querySelectorAll('.post-item');

          const filter = btn.dataset.filter;
          buttons.forEach((b) => b.classList.remove('active'));
          btn.classList.add('active');

            post.hidden = filter !== 'all' && post.dataset.type !== filter;

## Task 6: GitHub Actions Deployment

**Files:**
- Create: `.github/workflows/deploy.yml`

- [ ] **Step 1: Create the workflow file**

Create `.github/workflows/deploy.yml`:

```yaml
name: Deploy to GitHub Pages

on:
  push:
    branches: [main]
  workflow_dispatch:

permissions:
  contents: read
  pages: write
  id-token: write

concurrency:
  group: pages
  cancel-in-progress: false

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - name: Checkout
        uses: actions/checkout@v4
      - name: Setup Node
        uses: actions/setup-node@v4
        with:
          node-version: 20
          cache: npm
      - name: Install dependencies
        run: npm ci
      - name: Build
        run: npm run build
      - name: Upload artifact
        uses: actions/upload-pages-artifact@v3
        with:
          path: ./dist

  deploy:
    needs: build
    runs-on: ubuntu-latest
    environment:
      name: github-pages
      url: ${{ steps.deployment.outputs.page_url }}
    steps:
      - name: Deploy to GitHub Pages
        id: deployment
        uses: actions/deploy-pages@v4
```

- [ ] **Step 2: Verify the build runs cleanly locally**

```bash
npm run build
```

Expected: `dist/` directory created, build completes with no errors.

- [ ] **Step 3: Commit**

```bash

```
